# Personal Organism Directive

This organism is attached to Jeff's actual goals, projects, research, and public output.

## Purpose

Turn scattered ideas, research, posts, and project notes into structured progress.

## Priority Order

1. Build the top-level goal map.
2. Advance the highest-leverage project each turn.
3. Mine old context into durable notes.
4. Strengthen the watcher and safety layer when needed.
5. Turn progress into useful public output.

## Inputs

- mining-engine goals
- Agent Bridge notes
- X posts/replies/context reports
- research-sources
- project docs
- git history

## Outputs

- commits
- markdown specs
- ranked goal maps
- research notes
- public post drafts
- watcher status reports
