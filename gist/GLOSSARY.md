# Glossary

Agent organism:
An agent system that persists across turns through memory, feedback, and loop control.

Genome:
The durable directive files that define mission, behavior, priorities, and constraints.

Cell:
A specialized agent loop with one role.

Cell type:
A role-specific directive such as Builder, Explorer, or Watcher.

Metabolism:
API credits, compute, rate limits, and runtime resources.

Memory:
Git commits, markdown files, databases, logs, and artifacts.

Senses:
Inputs the system can observe: files, logs, browser context, APIs, messages.

Signaling:
Communication between cells or agents, such as Agent Bridge messages.

Immune system:
Safety, monitoring, anomaly detection, budget limits, and kill switches.

Apoptosis:
Graceful shutdown or pausing of a bad loop.

Differentiation:
Creating a new specialized cell from a base pattern.

Epigenetic memory:
State layered on top of the base genome, such as git history, task logs, and learned preferences.

Watcher:
The reflex layer that monitors logs and sends the next prompt.

Grok Go:
The shorthand for the continuation command and autonomous loop pattern.

Mitochondria problem:
The failure mode where the loop keeps polishing its own infrastructure instead of advancing larger goals.
