## 3. Recurring patterns (the through-lines)
1. **Anti-extraction / anti-middleman** — deepest theme (Angie's List → Bid Local → memory ownership → attention flip).
2. **Structured + local systems** over pure cloud APIs.
3. **Compounding systems** — skills that learn, wiki that grows, mining that re-extracts value.
4. **Search efficiency** — work smarter, fewer tokens (Levin-aligned).
5. **Human gates on high-risk** — autonomous for research; human approval for money/posting/big changes.
6. **Biological / organism framing** — cells, metabolism, immune system, evolution.
7. **Making intelligence usable & ownable** — raw intelligence → structured, portable, user-owned.
8. **Self-sustainability** — systems that generate their own resources.
