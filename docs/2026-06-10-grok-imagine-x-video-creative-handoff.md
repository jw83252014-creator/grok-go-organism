## The workflow (Grok Imagine → X)

1. **Image first.** Generate a strong still in Grok Imagine.
2. **Video second.** Re-run the same concept as a short 6–8s vertical/square clip.
3. **Post pattern that maximizes profile clicks:**
   - Post 1: strong image only, no link, short caption ("Most agents are fragile." / "One brain breaks. Cells adapt." / "A living system instead.")
   - Post 2 (self-reply): the video version.
   - Post 3: longer explanation or link to Grok Go.
4. **Improvement loop (Researcher Cell):** take a screenshot/description of current output → research what's better → write improved Grok Imagine prompts → short "why it's better" report → show the new visual. This loop is meant to run continuously and evolve the visuals like an organism.
