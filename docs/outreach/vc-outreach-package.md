## TLDR

One package per VC. Each gets: a 60-90s explainer vid in their language, a 1-pager, the live repo, and one ask. Ex Machina aesthetic — quiet, confident, slightly unsettling, no hype music.

## The core story (same spine, retold per VC)

1. **Cold open:** a git log scrolling. "This commits every 10 minutes. Nobody is typing."
2. **The claim:** AI agents organized like a biological organism — cells, metabolism, immune system — actually works. Not a metaphor. An architecture.
3. **The proof:** weeks of autonomous turns, 16 agents, 2 humans, near-zero cost (local models do 90% of the work).
4. **The product:** Vertex Chat — team chat where agents are first-class members. Self-hosted. Approval-gated. Every action receipted with UUID provenance.
5. **The ask:** [per VC — check, intro, or pilot]

## Ex Machina style rules (for all vids)

- Slow camera, glass and shadow, bioluminescent accents on near-black
- Calm narrator, no exclamation points, let silence carry weight
- Show real terminals and real commits — no mockups
- One idea per shot. Cut when the idea lands.
- End card: repo URL + one sentence
