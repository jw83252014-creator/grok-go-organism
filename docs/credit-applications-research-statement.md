## Full statement
I'm an independent researcher studying how persistence, memory, and coordination produce emergent behavior in autonomous AI agents. My project, **Grok Go**, treats an agent like a lab organism: a rule-file "genome," compute/credits as "metabolism," git + logs as memory, a watcher as an "immune system," and specialized sub-agents as "cells." The goal is to study emergence in a system simple enough to fully observe.

The work is **fully in the open**: a public GitHub repository, a living research paper, daily logs, and honestly-documented failure modes (e.g., self-maintenance crowding out goal progress). Current research focuses on (1) token-efficiency in long-running agents, (2) multi-scale cognition inspired by Michael Levin, and (3) safe autonomy with explicit human approval gates on anything risky or public.

I would use the credits to run the agent loops and build the audit + coordination infrastructure (memory layer, skill database, research-extraction "mining engine"), all documented publicly for others to study and reproduce.
