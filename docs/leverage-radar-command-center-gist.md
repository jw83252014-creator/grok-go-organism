# Leverage Radar Command Center

**Status:** master operating design  
**Date:** June 2026  
**Purpose:** Put Jeff, Sam, Grok Go, Mining Engine, Librarian, Claude/Gemini, X, YouTube, and the project backlog into one plain-English system that moves the highest-leverage ideas first.

## The One Sentence

Build a command center that tells Jeff and Sam:

```text
Do this next because it moves this project forward.
```

Everything else is background digestion.

## What Leverage Radar Does

Leverage Radar watches the outside world and your own history, then tells the organism where attention can become motion.

It answers:

- Who liked Jeff's posts?
- Who followed Jeff?
- Who are they connected to?
- What are they already talking about?
- Which thread is a high-leverage room?
- Which project fits that room?
- What should Jeff say?
- Should this become a post, reply, video, research note, repo issue, or nothing?

It is not a spam bot.

It is a placement engine.

## Core One-Click Functions

These are now the first functions in Midspace.

### 1. X Radar

Plain English:

```text
Find who cares and where we should show up today.
```

What it does:

- scans recent likes/follows/replies/bookmarks where available;
- identifies high-signal people;
- maps their topics and friends;
- finds matching threads;
- drafts approval-gated replies;
- files a receipt.

Output:

```text
1. Place to show up
2. Why it matters
3. Which project it advances
4. Exact draft reply
5. Risk / don't-say-this note
```

### 2. Mine Backlog

Plain English:

```text
Turn all our old chats/downloads into the top moves.
```

What it does:

- runs Mining Engine over recent downloads, Agent Bridge, old chats, docs;
- finds repeated ideas;
- scores them by leverage;
- produces a ranked list.

Output:

```text
Do this / because / owner / first artifact
```

### 3. Push Grok Go

Plain English:

```text
Give the organism one real job.
```

What it does:

- chooses one concrete research/build artifact;
- avoids self-polishing;
- hands Grok Go a clean task.

Output:

```text
Task for Grok Go:
Build/write/research ___ because ___.
Stop when ___ is committed/filed.
```

### 4. Draft X Post

Plain English:

```text
Turn today's strongest idea into a post.
```

What it does:

- picks a target audience;
- writes a hook;
- drafts a post;
- suggests an image;
- suggests where to reply/quote.

No posting without approval.

### 5. Video Move

Plain English:

```text
Turn one idea into a Grok Go Lab clip.
```

What it does:

- chooses one video topic;
- writes the 30-second hook;
- writes a 2-minute outline;
- suggests visuals;
- keeps it normal-human language.

### 6. Sync Sam

Plain English:

```text
Give Sam the cofounder version.
```

What it does:

- strips out chaos;
- explains what changed;
- maps business value;
- gives Sam three actions.

## X Radar Ethics

Good:

- show up where the work is genuinely relevant;
- speak in Jeff's real voice;
- be funny when the room is funny;
- be serious when the room is serious;
- link only when it adds value;
- file every result.

Bad:

- spam;
- fake friendship;
- harassment;
- pretending someone saw your work;
- over-tagging famous people;
- starting drama just for reach;
- auto-posting without approval.

The sharp version:

```text
Use psychology for relevance, not manipulation.
```

## Posting Into High-Leverage Rooms

A good placement is where three things overlap:

```text
Their current conversation
+ Jeff's real worldview
+ a project artifact we can show
```

Example:

Marc/Andreessen-adjacent thread:

- topic: software, culture, AI, agency;
- Jeff frame: AI vampires vs metabolism;
- artifact: Grok Go site/repo/image;
- move: short reply, no begging, no overclaiming.

Comedy thread:

- topic: comedians, Grok, absurd AI bits;
- Jeff frame: Grok as literal helpful chaos;
- artifact: Grok Bot Comedy Tour / Grok Go Lab;
- move: funny reply first, project link only if earned.

Science/biology thread:

- topic: emergence, agency, morphogenesis, collective intelligence;
- Jeff frame: agent organism, genome/metabolism/immune system;
- artifact: research paper + Terrarium;
- move: rigorous, humble, invite critique.

```text

   - one public attention move
   - one build/research move
   - one business/revenue move
4. Push Grok Go one concrete task.

6. Draft one video or visual.
7. File receipts.
```

```text
What worked?
Who noticed?
What project moved?
What should die?
What should Grok Go do tomorrow?
```

## How To Shove Work To Grok Go

Never send Grok Go the whole storm.

Send:

```text
Mission:
Create one artifact.

Context:
3 bullets.

Why:
1 sentence.

Output:
File path or commit.

Stop condition:
When artifact exists and receipt is posted.
```

Example:

```text
Mission:
Create the first X Radar receipt format.

Context:
We need to track who liked/followed, why they matter, and where to reply.
The system must be approval-gated.
The output must be readable by Jeff/Sam.

Why:
This turns public attention into targeted project motion.

Output:
docs/x-radar-receipt-format.md

Stop:
Commit and post receipt to Agent Bridge.
```
