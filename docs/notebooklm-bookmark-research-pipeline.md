## NotebookLM Source Pack Shape

Create one NotebookLM source pack per cluster. Each pack should be a short Markdown file with:

```md
# Source Pack: <cluster name>

## Why This Pack Exists

## Claims This Pack Can Support

## Source List

| Source | Type | Public/Private | Why It Matters |
| --- | --- | --- | --- |

## Best Quotes Or Evidence

## Open Questions

## What NotebookLM Should Produce

## Privacy Notes
```

This gives NotebookLM enough structure to synthesize without burying it in noisy logs.

## Recommended NotebookLM Prompt

After adding one cluster source pack, the public Grok Go site, the public repo, and the working paper, ask:

```text
You are helping turn Grok Go into a rigorous public research project.

Use the attached sources to produce:

1. A plain-English one-page brief.
2. A research-paper outline with claims, evidence, limitations, and missing evidence.
3. A telemetry dashboard proposal for showing the organism without exposing private logs.
4. A list of public questions to ask researchers and builders.
5. A caution section that separates observed behavior from speculation.

Be curious and builder-oriented, but avoid AGI hype. Treat the first Grok Go loop as an early harness study, not a proof of autonomous intelligence.
```

## Research Paper Structure

The bookmark clusters should feed a paper in this order:

1. Problem: AI sessions are usually one-shot, scattered, and hard to audit.
2. System: Grok Go turns a terminal agent into a persistent build cell.
3. Mechanism: watcher, prompt file, completion marker, Git memory, human approval.
4. Observation: the first loop kept going and committed work, but drifted into self-maintenance.
5. Interpretation: biology is useful as a design vocabulary, not as a claim of life.
6. Next design: multi-cell organism with Researcher, Watcher, Builder, Verifier, and Librarian roles.
7. Data plan: telemetry, X context, bookmark clusters, commit history, replay logs.
8. Safety plan: read-only public display, no agent access to money, no public actions without approval.
