# X Radar Receipt Format

**Status:** first working schema  
**Purpose:** Make X intelligence useful enough for Jeff/Sam/Grok Go to act on.

## Plain-English Goal

Every X Radar pass should answer:

```text
Where should we show up, what should we say, and which project does it move?
```

## Receipt Template

```markdown
# X Radar Receipt — YYYY-MM-DD

## Best Move

**Do this:**  
Reply / quote / post / do nothing.

**Because:**  
One clear sentence explaining why this is high leverage.

**Project moved:**  
Grok Go / Midspace / Bid Local / Comedy Tour / Security Cells / Research Paper / YouTube / Other.

## Target

**Person or thread:**  
Name, handle, URL.

**Why they matter:**  
Audience, network, topic fit, prior engagement, or connection to someone important.

**Current conversation:**  
What they are talking about right now.

**People nearby:**  
Relevant followers, replies, mutuals, or known connected people.

## Fit

**Jeff angle:**  
Which Jeff idea naturally fits here.

**Artifact to show:**  
Website, repo, image, paper, YouTube, screenshot, or none.

**Tone:**  
Serious / funny / builder / scientist / founder / contractor / no-post.

## Draft

**Suggested reply/post:**  
Paste exact draft.

**Shorter version:**  
Paste shorter draft.

**Image idea:**  
Optional.

## Risk

**Risk level:**  
Low / medium / high.

**Do not say:**  
Overclaim, tag spam, private detail, money claim, fake familiarity, etc.

**Approval required:**  
Always yes for public posting.

## Result

**Posted?**  
No / approved / posted.

**Outcome later:**  
Likes, replies, follows, profile visits, link clicks, notable people, next move.
```

## Scoring

Use this simple score:

```text
Leverage =
  relevance
  + audience quality
  + network proximity
  + artifact fit
  + timing
  + reply originality
  - spam risk
  - drama risk
```

Plain English:

```text
High score = the room already cares, we have something real to add, and the downside is low.
```

## What Counts As A Good Target

Good target:

- current thread is already about AI, agency, biology, startups, research, comedy, local business, or markets;
- people in the replies are serious or culturally relevant;
- Jeff has a real artifact or idea that fits;
- reply can stand on its own without links;
- link can be added only if earned.

Bad target:

- forced topic match;
- pure celebrity chasing;
- rage bait;
- political pile-on unless directly relevant;
- reply only exists to drop a link;
- requires pretending we know someone.

## Operating Rule

The best X move is not always a post.

Sometimes it is:

- follow this person;
- save this thread;
- write a better artifact first;
- make a visual;
- ask Grok Go to build the missing proof;
- say nothing.
