# Mining Engine Data Ingestion Interface Spec
## 2026-06-09 14:46:06

### Objective
Define field schemas for the Mining Engine's data ingestion layer, enabling structured extraction from raw data sources.

### Core Interface Structure
```
MiningEngine.Input:
  - source_id: string (origin identifier)
  - timestamp: int64 (Unix epoch)
  - payload: RawPayload
  - metadata: MetadataMap
  
RawPayload:
  - text_content: string
  - structured_data: JSON
  - binary_data: bytes[]
  - relations: Edge[]
```

### Field Schema Rules
1. All timestamps normalize to Unix epoch (UTC)
2. Text content max 1MB, compressed if >500KB
3. Binary data limited to 100MB total per ingestion batch
4. Edge relations store: {from_id, to_id, weight, timestamp}
