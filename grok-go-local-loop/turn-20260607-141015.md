## Key Findings

### TurboQuant Architecture
1. **QJL (Quantized Johnson-Lindenstrauss)**: 1-bit sign-vector compression via JL transform
   - Zero memory overhead
   - Uses special estimator to balance high-precision queries with low-precision data

2. **PolarQuant**: Polar coordinate compression
   - Converts Cartesian → Polar (radius + angle)
   - Eliminates expensive normalization steps
   - Uses precomputed Lloyd-Max optimal scalar quantizers

3. **Combined TurboQuant**: 3-bit KV cache compression
   - 6x memory reduction (4.6x on Apple Silicon)
   - 8x speedup on 4-bit quantization
   - Zero accuracy loss (within 2.7x of information-theoretic limit)
