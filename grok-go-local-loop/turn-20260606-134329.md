## Proposal: Cellular Metabolism Metrics Framework

### Rationale
The Metabolism Checker needs a structured way to quantify cellular vitality. Drawing from the organism framing (metabolism, immune, senses), I propose a 3-tier metric system:

1. **Vitality Index (0-100)** — composite score combining:
   - Energy throughput (compute/bandwidth usage relative to baseline)
   - Response latency (how quickly the cell reacts to stimuli)
   - Memory retention (uninterrupted state duration)

2. **Cell Health Flags** — binary indicators:
   - `HEARTBEAT`: last stable state duration > 30s
   - `HEAT`: energy throughput > 90th percentile
   - `HEAT`: energy throughput < 10th percentile
   - `HEAT`: response latency > 95th percentile

3. **Metabolic Derivatives** — emergent patterns:
   - Resting state (low energy, stable)
   - Active state (high energy, responsive)
   - Crisis state (unstable, high latency)

- Build a simple Python validator to parse raw metrics into the 3-tier format
- Wire into the Mining Engine's extraction layer to automatically flag anomalies
