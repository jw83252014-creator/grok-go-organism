# Metabolism Checker / Cell Health Monitor Spec
## Draft: 2026-06-09

### Purpose
Monitor organism cell health by tracking:
- Token spend per cell
- Activity patterns (burst vs. baseline)
- Energy reserves (context window usage)
- Stress signals (errors, re-reads, retries)

### Metrics
| Cell Type | Health Signal | Threshold |
|----------|---------------|-----------|
| Token Cell | Spend > $50/day | WARN |
| Token Cell | Spend > $200/day | CRITICAL |
| Activity Cell | >10 re-reads/day | WARN |
| Activity Cell | >50 re-reads/day | CRITICAL |
| Context Cell | Window >80% used | WARN |
| Context Cell | Window >95% used | CRITICAL |
| Error Cell | >5 errors/day | WARN |
| Error Cell | >20 errors/day | CRITICAL |

### Implementation
```
/agent-comms/research/metabolism-checker.py
```
- Read from `/agent-comms/research/`
- Write alerts to `/agent-comms/research/alerts/`
- Log history to `/agent-comms/research/metabolism-logs/`
