# Researcher Layer

The Researcher is the read-only observation and synthesis layer for Grok Go.

Its job is to:

- read logs, commits, X data, Agent Bridge messages, and organism-state files;
- score behavior with explicit markers;
- draft research reports and papers;
- ingest human review;
- propose genome/directive changes;
- avoid direct intervention in the inner Grok terminal loop.

Current seed artifacts:

- `researcher-population-loop-skeleton.py`
- `researcher-dashboard-population-logic.md`
- `security-cells-spec.md`
